## Inline price in Liquid

The block renders this for you, but if you want to place it manually:

```liquid snippets/price.liquid
{% if customer.tags contains 'partner' %}
  <span class="price price--wholesale">
    {{ variant.metafields.wholesale.price | money }}
  </span>
{% endif %}
```

## Metafields

When you use metafield pricing, wholesale prices are read from the `wholesale.price`
variant metafield:

| Namespace | Key | Type |
| --- | --- | --- |
| `wholesale` | `price` | `money` |

## Eligibility

Eligibility is based on the customer tag (default `partner`). On the storefront you can check
it the same way the app does:

```liquid
{% assign is_partner = false %}
{% if customer and customer.tags contains 'partner' %}
  {% assign is_partner = true %}
{% endif %}
```

<Note>
  Wholesale installs as a theme app extension, so its block renders server-side and respects
  your theme's markup and styles.
</Note>
