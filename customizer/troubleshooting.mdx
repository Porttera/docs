---
title: "Troubleshooting"
description: "Common Product Customizer issues and how to fix them."
---

<AccordionGroup>
  <Accordion title="The designer doesn't appear on a product" icon="eye-slash">
    - Confirm the **Customizer block** is added to that **product template** in the theme editor.
    - Make sure the product is assigned the template the block is on.
    - Save and republish the theme after adding the block.
  </Accordion>
  <Accordion title="The preview doesn't match the print area" icon="crop">
    Tighten the **printable bounds** for each decoration area in the app admin so the preview
    matches your real production area.
  </Accordion>
  <Accordion title="Artwork isn't on the order" icon="image">
    - Confirm the shopper actually **uploaded** a file (not just added text).
    - Check the file was within the **allowed type and size** limits.
    - Open the order — artwork is attached as line-item properties and stored in **Shopify Files**.
  </Accordion>
  <Accordion title="A customer can't find their saved design" icon="floppy-disk">
    Saved designs require the customer to be **signed in** to their account. Guests can
    customize but can't save.
  </Accordion>
  <Accordion title="An upload is rejected" icon="ban">
    The file is likely the wrong **type** or over the **size limit**. Adjust the allowed types
    and max size in **Settings → Uploads**.
  </Accordion>
</AccordionGroup>

<Note>

  screenshot of what you're seeing.
