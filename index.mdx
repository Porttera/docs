## How the apps install

<Card title="Theme app extensions" icon="puzzle-piece" href="/install">
  Every Porttera app installs as a Shopify theme app extension — a block you add in the
  theme editor. No code is injected into your theme, and uninstalling leaves nothing behind.
</Card>
